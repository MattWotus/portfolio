Healthy Habitation

A full-stack React.js and Node.js shopping cart application.

Features

- User can view the products for sale.

- User can view the details of a product.

- User can add a product to their cart.

- User can view their cart summary.

- User can place an order.

Technologies Used

React.js | Node.js | Express.js | PostgreSQL | Webpack | Babel | Pgweb | JavaScript ES6 | Bootstrap 4 | HTML5 | CSS3
