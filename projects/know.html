Know

A full-stack React.js and Node.js application designed to help lower the rate of new HIV and STD diagnoses in the United States, allowing sexually-active people to have a central location to keep track of their sexual health, and specifically, HIV/STD test results.

About

This project was a group project, and the features mentioned make up the MVP of the project for the time given to create it. There are lots of opportunties for stretch features and improvements, and I look forward to continuing to work on the application.

My Roles: Team Lead, Front-End Lead, Full-Stack Developer, Principal Designer

To try the application live, click on the View Project button, then adjust the width of your browser window by clicking on the edge of the window and dragging inward until you're unable to anymore. If you're using Google Chrome, instead of resizing your browser, you can right click, select Inspect, click the phone/tablet icon in the top left corner of the Chrome Dev Tools window, and choose a phone to view the application on.

All Features

- User can view all visits.

- User can view test results from a single visit.

- User can add a visit.

- User can delete a visit.

- User can view all partners.

- User can view details from a single partner.

- User can add a partner

- User can serach for places to get tested.

Features I Worked On

- User can view all visits, front-end.

- User can view test results from individual visit, front-end.

- User can add a visit, front-end.

- User can delete a visit, front-end.

- User can navigate in app via navbar, front-end.

- User can search for places to get tested, front-end.

- User can view all partners, back-end.

- User can add a partner, back-end.
