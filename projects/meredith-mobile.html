Meredith Mobile

Meredith College's Mobile Homepage

A redesign and redevelopment of Meredith College's mobile homepage.

Features

- Users can view and scroll down the redeveloped mobile homepage of Meredith College's website.

- Users can click the menu icon to view the main navigation dropdown menu.

- Users can click on the person icon to view the secondary navigation dropdown menu.

Technologies Used

JavaScript ES5 | Flexbox | HTML5 | CSS3
