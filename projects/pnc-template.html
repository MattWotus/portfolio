PNC Template

PNC Corporate PAC Website Template

An application template (home page and one interior page) designed and built for internal use at PNC Bank to serve as the company's corporate super PAC website.

Technologies Used

JavaScript ES5 | jQuery | Maps JavaScript API | Bootstrap 4 | Flexbox | HTML5 | CSS3 | Adobe Illustrator

*Note: For this project, I built a home page and interior page template for PNC Bank, so there are only two active pages when the project is viewed (the home page and about page).
