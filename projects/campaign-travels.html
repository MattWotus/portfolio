Campaign Travels

Hillary Clinton's Campaign Travels

A JavaScript application that displays a map of Hillary Clinton's campaign travels from October 2, 2016, through Election Day, with markers and event details for each city.

Features

- Users can view map of Hillary Clinton's campaign travels from October 2, 2016, through Election Day.

- Users can view markers for each campaign stop, with each marker containing a number representing the stop number during the indicated time period.

- Users can click on a marker and view the details of the stop, the miles Clinton traveled from the previous stop, and the total number of miles traveled to date during the indicated time period.

- Users can click on a marker cluster, which represents a city Clinton made more than one stop in during the indicated time period, to view the separate stops.

Technologies Used

JavaScript ES5 | jQuery | Maps JavaScript API | Bootstrap 4 | Flexbox | HTML5 | CSS3
